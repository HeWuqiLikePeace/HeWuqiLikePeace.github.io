在树莓派上部署私有云（学习记录）

2022-1-26

看到了树莓派爱好者基地推送的文章，又正好放寒假，

那就搞一搞呗，按教程一步步走就完事了，

理应是这样的，

在安装管理客户端时出现了以下提示：

    为了让这个仓库能够应用，这必须在更新之前显式接受 

（别说我打错字了，这是Linux系统汉化问题）

解决此故障可将：

   sudo apt-get update 

更改为：

   sudo apt update     

并在出现提示时选择“Y”即可正常更新安装

最后在同一局域网下访问192.168.0.12:8888进入登录页面

重点：在每次启动树莓派时要进入终端启动服务

   docker start nextcloud    #启动服务   

     docker restart nextcloud  #重启服务       

   docker stop nextcloud     #停止服务   

（然后发现树莓派存储空间太小不能满足长期存储需要，这个项目宣告结束）

附件：docker安装nextcloud.txt
